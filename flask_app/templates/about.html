About this project

Spotify Crescendo is a project done in Spring 2020 for Cornell Data Science Insight's Project Team.

The project focused on analyzing the top charts of Spotify's songs from the past couple of years and aimed to identify characteristics from those top songs.

The data for this project was gathered through Spotify's API. First we web scraped the top 200 songs from Jan 2018 - Feb 2020 using this website. Then, using python and the spotipy library we gathered all the spotify features for each song.

We also used the Spotify API to get artist features, such as popularity, number of followers, and their genres.

Using this data, we used machine learning techniques to try to predict some things from the data. We built algorithms to predict the number of days a song would stay in the top charts, an algorithm to capture the overall trend of the song, and a clustering technique to try to predict the genre of a song.
